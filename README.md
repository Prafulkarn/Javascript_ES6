# Lab3 JavaScript Practice

Small JavaScript lab project with ES6 exercises and DOM mini-apps.

## What is included

- ES6 practice in basics.js
- Dynamic Table (add/delete rows)
- Dynamic List (add/remove items)
- Light Toggle (bulb ON/OFF)
- Password Toggle (show/hide)
- Registration Form (client-side validation)

## Run locally

1. Open this folder in VS Code.
2. Open any HTML file in a browser.
3. Or use Live Server for quick preview.

Entry files:

- Dynamic table/index.html
- DynamicList/list.html
- light toggle/index.html
- passwordHide/index.html
- RegistrationForm/index.html

## Tech

HTML, CSS, JavaScript (ES6)
